Table 3 of 3 - Summary of Control Change Messages (Data Bytes)

The following table lists the Control Change messages in numerical (binary) order. This table is intended as a quick reference to all Control Change and Mode Change messages. 

Appended at the bottom is a table of registered parameter numbers (sort-of extensions to Control Change message).

See also:

  • Table 1 - Summary of MIDI Messages
  • Table 2 - Expanded Messages List (Status Bytes)

Additional messages are listed in the printed documentation available from the MMA.

compatibility with other products. We strongly recommend consulting the official
 MMA Detailed MIDI Specification for additional information.
